# RampApp Mobile 


## Dependencias 
* NodeJs 21 
* Una vm con android y/o IOS o un celular con la app de expo (puede escanear el qr para correrlo).
## Para instalar dependencias
### `npm install` 

## Para correrlo
### `npx expo start`
